**Цикломатска комплексност**

Цикломатската комплексност на овој код е 8. Тоа го добив по формулата ( А - В + 2 ), каде А е бројот на ребра а В е бројот на темиња/јазли. 
Во овој случај добив ( 25 - 19 + 2 = 8 )
